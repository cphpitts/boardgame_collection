Welcome to the Boardgame Information Collector

The Boardgame Information collector allows users to track their boardgame collection by either manually inputting information or using an API to add games listed on BoardGameGeek's list of hot games. This app also features a news section that uses a web scraper to take the most recent articles from DiceTowerNews.com.

Technologies: HTML, CSS, Javascript, Python, Django

API Information: https://boardgamegeek.com/wiki/page/BGG_XML_API2
